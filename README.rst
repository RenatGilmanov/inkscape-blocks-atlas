============================
Atlas block library
============================

-----------
Description
-----------
Simplified isometric blocks to help visualize network topologies, deployment strategies, etc.

Initially created by `AterLeo`_

.. _AterLeo: http://leo-tvorit.lj.ru

--------
Features
--------
- modular design
- basic shapes and colors
